### 创建仓库

- **git init**

```js
git init //在当前目录下生成一个.git目录
git init newrepo //在newrepo目录下生成一个.git目录
```

- **git clone**

``` js
git clone <路径>//从指定路径中拷贝项目
```

> ``` 
> //例
> git clone git@github.com:1118gmh/git-.git 
> ```
>
> 执行完该命令后会在当前目录下创建一个git-的目录，其中包含.git的目录，用于保存下载下来的所有版本记录。
>
> 如果要自己定义项目目录名称，则在末尾添加新的名字即可
>
> ```
> git clone git@github.com:1118gmh/git-.git mygit学习
> ```
