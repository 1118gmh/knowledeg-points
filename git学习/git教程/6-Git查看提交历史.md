### Git查看提交历史

> 查看提交的历史

```js
git log 
```

> 查看历史记录的简洁版本

```js
git log --oneline
```

> 查看历史中什么时候出现了分支、合并

```js
git log --graph
```

> 逆向显示所有日志

```js
git log --reverse
```

> 查找指定用户的提交日志

```js
git log --autor=<用户名>
```

> 查找指定日期的提交日志

```js
git log --before={3.weeks.ago} --after={2010-04-18}//3个月前4月28日之后的所有提交
```
